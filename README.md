# snowball

Snowball is an easily scriptable IRC bot written in PHP.

## Why The Name Snowball?

```
irc
icee

```

## How Do I Install It?

```bash
git clone https://github.com/lukevers/snowball
cd snowball
composer dump-autoload
```
